# SoundSource — Portable System Audio Control with Per‑App Volume

## About SoundSource

SoundSource provides a complete system audio control environment for users who want per‑app volume management and global effects. SoundSource combines per‑app volume sliders, system equalizer, audio balance, and effects. SoundSource enables users to control volume of individual apps, boost quiet apps, and apply EQ across all audio. SoundSource runs on desktop platforms. SoundSource is trusted by podcasters, streamers, musicians, video editors, and audio enthusiasts worldwide.

## Deep Dive — System Audio Control Notes

Per‑app volume: individual volume slider for each running app
Equalizer: 10‑band and 31‑band EQ, presets (Rock, Jazz, Classical)
Balance: left/right balance adjustment, mono audio toggle
Effects: volume boost, compressor, limiter, gain, sound enchancer
Output device: per‑app output routing, system output selection
